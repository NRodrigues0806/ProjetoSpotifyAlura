# spotify-imersao
 Projeto de Streaming de Música
Este projeto é uma aplicação web desenvolvida durante a Imersão Front-end da Alura, onde criamos uma interface inspirada em um serviço de streaming de música. A aplicação utiliza HTML, CSS e JavaScript, além da biblioteca React para gerenciar a interface de forma dinâmica e interativa.

Funcionalidades
Listagem de Artistas: A aplicação exibe uma lista de artistas, permitindo que os usuários visualizem informações sobre cada um.
Busca Interativa: Os usuários podem buscar artistas específicos através de uma barra de pesquisa.
Componentização: O projeto foi estruturado utilizando componentes React, facilitando a manutenção e escalabilidade da aplicação.
Tecnologias Utilizadas
HTML
CSS
JavaScript
React
JSON Server (para simular uma API)
